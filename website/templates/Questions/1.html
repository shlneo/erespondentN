Как зарегистрироваться юридическому лицу?

Находясь на главной странице, для регистрации необходимо перейти по клавише «Зарегистрироваться», для перенаправления на страницу создания учетной записи (см. рисунок 1).

Рисунок 1 - Клавиша «Зарегистрироваться»

В форме регистрации потребуется ввести:

  • Электронная почта
  • Пароль
  • Подтверждение пароля
  • На один адрес электронной почты можно создать только один аккаунт

Также необходимо принять условия пользовательского соглашения (см. рисунок 2).

Рисунок 2 - Форма регистрации

После успешного заполнения данных и клика на клавишу «Зарегистрироваться», пользователь должен подтвердить ранее указанную электронную почту, путем ввода проверочного кода (отправленного на почту) (см. рисунок 3).

Рисунок 3 - Ввод проверочного кода

В случае необходимости отправить код ещё раз, можно воспользоваться клавишей «Отправить еще раз». После успешной проверки электронной почты пользователь будет переведен на страницу авторизации где можно будет авторизоваться в только что созданую учетную запись (см. рисунок 4).
