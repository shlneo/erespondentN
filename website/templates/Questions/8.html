Особенности заполнения формы отчета

Изначально пользователю доступно 3 вида продукции (см. рисунок 15):

  • Всего по нормированному потреблению — сумма всей добавленной продукции по каждому разделу, не включая Прочее потребление.
  • Прочее потребление — не входящее в список потребление, добавленное пользователем.
  • Всего потребление (сумма строк 9001, 9010) — сумма всей добавленной продукции по каждому разделу, включая Прочее потребление.
Рисунок 15 - Основные виды продукции
  • Строки с кодом строки 9001, 9010, 9100 не подлежат удалению.

В строке с кодом 9010 отражаются данные о расходе энергоресурсов на прочее (ненормируемое) потребление и подводится итог о расходе соответствующих энергоресурсов на производство всех видов продукции (работ, услуг), включая расходы на прочее потребление.

Для указания такой продукции необходимо кликнуть правой кнопкой мыши по строке «Прочее потребление» и выбрать пункт «Редактировать» в появившемся контекстном меню. В открывшемся модальном окне следует заполнить поля: [5] Израсходовано на всю производственную продукцию фактически и Примечание (указать вид продукции). После заполнения нажмите кнопку «Изменить» для сохранения изменений (см. рисунки 16–17).

Рисунок 18 - Особое заполнение

При заполнении кода строки 7000 не требуется указывать значения для следующих параметров (см. рисунок 19):

  • [1] Произведено продукции (выполнено работ, услуг) за отчётный период;
  • [2] Утверждённая норма на единицу продукции;
  • [3] Израсходовано на единицу продукции фактически.
